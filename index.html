Pokémon Pixel Canvas

Site recréant les pokémons de la première génération pixel par pixel en utilisant HTML, CSS et Javascript.
J'ai utilisé les canvas afin de lire l'image source et reprendre le code couleur de chaques pixels. Ensuite je colore un élément HTML correspondant à chacun de ces pixels.
L'utilisateur n'a qu'à écrire le nom du pokémon dans la barre de recherche pour que celui là s'affiche ( en français ou en anglais ).
